# AssistX Monitoring Android App

AssistX Monitoring is an Android application designed for real-time monitoring of PC clients and projects. This repository contains the source code and the latest releases for the Android application.

## Features
- Real-time client monitoring and status updates
- Project-based grouping of clients
- Alert management and notification system
- Push notifications via SSE
- Performance dashboard and history logs

## Installation
You can download the latest APK from the Releases section of this repository.

1. Go to the Releases page.
2. Download the AssistXMonitor-v2.apk file.
3. Install the APK on your Android device. Make sure to allow installation from unknown sources if prompted.

## Requirements
- Minimum Android SDK version: 24 (Android 7.0)
- Target Android SDK version: 34 (Android 14)

## Architecture and Technologies
This app is built using Kotlin and follows modern Android development practices:
- MVVM Architecture
- Kotlin Coroutines for asynchronous operations
- Retrofit and OkHttp for network requests
- Server-Sent Events (SSE) for real-time updates

## Development Setup
To build the project locally:
1. Clone this repository.
2. Open the project in Android Studio.
3. Sync the Gradle files.
4. Build and run the app on an emulator or a physical device.
